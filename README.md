# A股周赛红榜

## 页面

- `index.html`：展示页，只显示本周排名、每周回溯、规则。
- `admin.html`：管理页，用于提交参赛、同步行情、结算、保存回溯。

## 同步行情

1. 双击 `sync_quotes.bat`。
2. 首次运行会自动安装 `akshare` 和 `pandas`。
3. 输入本周起始交易日和结束/当前日期，例如：
   - 起始交易日：`20260420`
   - 结束日期：`20260424`
4. 脚本会把行情写入 `data.js`。
5. 打开或刷新 `admin.html`，点击“用数据文件刷新页面”。
6. 检查排名无误后，点击“保存本周回溯”。

## 数据口径

- 周一开盘价来自本周第一个有效交易日开盘价。
- 结算价默认优先使用实时最新价；运行脚本时加 `--no-realtime` 可只用历史日线最后一个收盘价。
- 周一开盘涨停会按前一交易日收盘价估算涨停价并自动标记。
- ST 名称会自动标记为无效。

## 重要说明

静态网页无法直接稳定调用行情源，所以行情由 `sync_stocks.py` 获取并写入 `data.js`。发布给别人手机查看时，需要把 `data.js` 一起上传。
